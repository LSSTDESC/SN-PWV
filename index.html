Impact of Chromatic Effects on LSST SNe Ia¶

The Supernova Atmospheric Simulation package (snat_sim) was developed within the Dark Energy Science Collaboration (DESC) to support research into the impact of atmospheric calibration techniques on astronomical observations of Type Ia Supernovae (SNe Ia). This documentation is provided as a technical reference for the package as well as minimal outline of the research it was designed to support. For questions concerning the snat_sim package, please see the API section of these docs or raise an issue on GitHub.

Pipeline Overview¶

The snat_sim package is designed first and foremost to support scientific research efforts. The documentation may at times assume the reader has a basic understanding of astronomy and how astronomical surveys work. However, these assumptions are kept to a minimum wherever possible.

snat_sim is designed such that each module represents a distinct stage in the research process. These steps range in complexity from visualizing simulation results to simulating the brightness of astronomical objects as a function of time. Ultimately each of these tasks are combined to support a parallelized analysis pipeline that handles the bulk of simulation process (outlined below). Outputs from this pipeline are then reviewed by researches to draw scientific conclusions.
